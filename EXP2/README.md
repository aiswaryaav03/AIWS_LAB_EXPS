# Online Event Registration & Promotional Portal
## Experiment 2: Advanced CSS Styling

### Project Overview
This project enhances a basic HTML5 portal using **Advanced CSS3** techniques including Flexbox, Transitions, and Media Queries for a fully responsive user experience.

### Key Features
- **Responsive Layout:** Adapts to Desktop, Tablet, and Mobile using Media Queries.
- **Flexbox Grid:** Side-by-side alignment of media and registration forms.
- **Interactive UI:** Smooth hover transitions on buttons and focus effects on input fields.

### Technologies Used
- HTML5
- CSS3 (Flexbox, Transitions, Media Queries)
